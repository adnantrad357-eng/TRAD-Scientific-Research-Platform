# 🎓 منصة البحث العلمي - طلاب كلية الاقتصاد

منصة متكاملة لإنشاء الأبحاث الأكاديمية وتحليل البيانات الإحصائية للدراسات الاقتصادية.

## 🚀 المميزات

### 1. إنشاء بحث علمي
- معالج خطوة بخطوة (6 خطوات)
- إدارة الأسئلة والفرضيات والأهداف
- اختيار المنهجية واللغة ونمط التوثيق
- إضافة محتوى متقدم (معادلات، جداول، دراسات حالة)

### 2. تحليل البيانات
- حساب المؤشرات الإحصائية (متوسط، وسيط، انحراف معياري...)
- رسوم بيانية تفاعلية (Chart.js)
  - السلسلة الزمنية + المتوسط المتحرك
  - التوزيع التكراري (Histogram)
  - Box Plot
  - الانحدار الخطي + التنبؤ
- جدول تحليلي مفصل مع Z-Scores
- تفسير علمي ذكي
- تصدير CSV و PDF

### 3. اختبارات إحصائية
- ارتباط بيرسون مع scatter plot
- اختبار T المستقل
- ANOVA (تحليل التباين)
- كاي تربيع (Chi-Square)

### 4. الاستبيانات
- بناء أسئلة JSON
- تحليل ردود CSV
- دعم المقابلات النوعية
- تقارير تفاعلية

### 5. السلاسل الزمنية
- مجموعات بيانات جاهزة (سوريا، تركيا)
- جلب بيانات تلقائي
- رسم بياني تفاعلي
- إرسال مباشر لقسم التحليل

### 6. تكامل KoboToolbox
- ربط API Token
- جلب النماذج والردود
- تحليل تلقائي

### 7. لوحة تحكم Admin
- إحصائيات عامة
- إدارة المستخدمين
- مراجعة الأبحاث
- إعدادات المنصة

## 🌐 النشر على Firebase Hosting

### الخطوة 1: تثبيت Firebase CLI
```bash
npm install -g firebase-tools
```

### الخطوة 2: تسجيل الدخول
```bash
firebase login
```

### الخطوة 3: إنشاء مشروع
```bash
firebase init hosting
```
- اختر "Create a new project"
- اختر اسم المشروع
- اختر المجلد العام: `economics-research-platform`
- اختر "No" للـ SPA

### الخطوة 4: النشر
```bash
firebase deploy
```
